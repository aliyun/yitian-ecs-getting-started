# 介绍

容器服务 Kubernetes 版（简称 ACK）提供高性能可伸缩的容器应用管理能力，支持企业级容器化应用的全生命周期管理；2021年成为国内唯一连续三年入选Gartner公共云容器报告的产品，2022年国内唯一进入Forrester领导者象限。其整合了阿里云虚拟化、存储、网络和安全能力，助力企业高效运行云端Kubernetes容器化应用。
